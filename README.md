# PHY564A Group project

Stepmania based game project for PHY564A Open Electronics course at Ecole polytechnique.

## ESP32 pinout for the controllers

| Sig1 Pins(LED inputs)    | Sig2 Pins(Button outputs) |
| ------------------------ | ------------------------- |
| 26 (D3)  | 25 (D2)   |
| 13 (D7)  | 14 (D6)   |
| 16 (D11) | 17 (D10)  |
| 12 (D13) | 4 (D12)   |
